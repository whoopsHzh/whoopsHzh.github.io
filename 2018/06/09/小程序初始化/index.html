mpvue小程序初始

Veroffentlicht am 2018-06-09 | Bearbeitet am 2019-04-14 | in mpvue

美团都开源了mpvue，由于mpvue框架是完全基于Vue框架的（重写了其runtime和compiler），因此在用法上面是高度和Vue一致的（某些功能由于受限于小程序环境本身的原因而不能使用），这给使用过Vue开发Web应用的前端开发者提供了极低的切换门槛来开发小程序。

mpvue小程序初始

环境

点击下载node.js，安装很简单，直接撸下一步

$ node -V

成功安装会输出版本号

安装完可以把npm切换国内淘宝源

npm set registry https://registry.npm.taobao.org/

安装vue-cli

npm install vue-cli -g

完成时：

vue

会出现提示命令

微信开发工具

微信的开发调试工具，安装微信开发工具

创建基于mpvue的小程序

vue init mpvue/mpvue-quickstart firstapp(项目名)
接下来引导配置，跟着撸就OK了。用vue 的语法，用wx接口。爽歪歪
